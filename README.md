 About
FoodLink is a community-driven platform designed to reduce food waste and fight hunger. Donors, volunteers, and delivery partners can collaborate to ensure surplus food reaches those in need efficiently.

---

 Features
- Three types of users: Donor, Volunteer, Delivery Partner  
- Hunger Pins: Locate nearby hunger spots easily  
- Real-time Tracking: Track your food donation status  
- User-friendly Dashboard: View detailed actions for each user type  
- Notifications: Alerts for nearby donations or volunteer requests
